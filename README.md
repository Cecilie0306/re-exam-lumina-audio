# Landing Page
Lavet af Cecilie Andreasen, MDU25E
Der er blevet lavet små ændringer/redigeret i html og i css, så layoutet er bedre at se på. 
Der er blevet tilføjet en mappe med js og en javascript-fil. 

I html er der blevet lavet nye sections og blevet tilføjet et flip-card på hver section. 
I css er der blevet tilføjet animationer så billederne kan bevæge sig. 

De største ændringer:
1. Sections
2. Flip-cards
3. JS
4. Se Produkter-knap
5. Animationer
